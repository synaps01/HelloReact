# HelloReact
React Test App

## About HelloReact
This is a ReactJS app that follows the Udemy course on https://www.udemy.com/the-complete-react-web-app-developer-course

## Build Requirements
+ Node 5.5.0
+ npm 3.3.0

## Modules
+ "express": "4.15.2"

## How to run

+ Clone the project
+ Run npm install
+ Run '''node server.js'''
